# Telecom-insight-Genrator
An AI-powered telecom analytics platform that enables users to query and analyze telecom network data using natural language. The application leverages Spring AI, RAG (Retrieval-Augmented Generation), Semantic Search, and Gemini LLM to convert user queries into SQL, generate business insights, and provide real-time network intelligence.
